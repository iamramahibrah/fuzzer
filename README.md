# Fuzzer 
simple fuzzing tool inspired by ffuf 

### Installation
````
git@github.com:iamramahibrah/fuzzer.git
./fuzzer.sh --install
````
### Usage
````
./fuzzer {options}
         -H --help    | Show usage help
         -i --install | Install tools
         -c --check   | check tools
         -d --dir     | Discover hidden directories /admin panel 

````
#### Tools
- ffuf
- golang(go)
- Amass
